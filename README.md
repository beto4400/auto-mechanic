# AutoFix Pro — Premium Landing Page

Expert auto repair services landing page built with Next.js.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Deploy

Deploy with Vercel:

```bash
vercel
```

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Lucide Icons** - Icon library

## Project Structure

```
app/
  layout.tsx        # Root layout
  page.tsx          # AutoFix Pro landing page
  globals.css       # Global styles
```
